# Context

This folder contains living documents that guide development of LabRat.

Claude Code should read all files in this folder at the start of every session.

| File | Purpose |
|---|---|
| `ai-guidelines.md` | Workflow, commit rules, and how to interact with Claude Code |
| `current-feature.md` | The active feature being built right now |
| `coding-standards.md` | Kotlin, Compose, and architecture rules |
| `project-schema.md` | Full project roadmap broken into phases |
| `specs/` | Upcoming and in-progress feature specs |
| `features/` | Archive of completed feature docs |
